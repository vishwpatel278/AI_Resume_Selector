# AI_Resume_Selector
AI-powered resume screening system that parses resumes and ranks candidates based on job requirements using Node.js, Express, and MongoDB.

This project is an AI-powered Resume Selector backend that automates the process of screening and ranking candidates. It allows users to upload resumes, extracts relevant information such as skills, experience, and education, and matches candidates against job requirements using a scoring algorithm.

Key Features:
- Resume parsing and data extraction
- Intelligent candidate ranking based on job descriptions
- RESTful APIs for resume upload and retrieval
- MongoDB integration for scalable data storage
- Secure and efficient backend architecture

Tech Stack:
- Node.js
- Express.js
- MongoDB
